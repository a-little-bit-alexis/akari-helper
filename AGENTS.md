# AGENTS.md

This is a React, TypeScript, and npm project for a personal Akari puzzle helper, called `akari-helper`.

Use `./scripts/check.sh` after code changes. It builds the app, runs ESLint, checks Prettier formatting, and runs Jest.

Do not add ARIA attributes or ARIA roles in this project. The app is a visual puzzle game intended for personal use, and the current project preference is to keep components free of ARIA-specific markup.

Each React component should have its own file. Do not write tests for React components.

# Purpose

The purpose of akari-helper is to perform a variety of functions related to the puzzle game Akari. At a high level, Akari is a binary-determination logic puzzle, a constraint satisfaction problem. A player is presented with a puzzle, either generated by akari-helper or from an external source (a book, another app), and tries to solve it.

When performing a task where you need to know the rules of Akari, please read Akari.md.

Eventually, we want the user to be able to:

- Play Akari (place marks on the board; perform a series of deductions to find a solution) within the akari-helper app
- Ask akari-helper to find a solution
- Ask akari-helper to perform a single deduction and explain it
- Ask akari-helper for the logically simplest next move (i.e., if the user is stuck and doesn't want to just resort to brute-force guessing or hypothesis-testing)
- Ask akari-helper to generate a puzzle
- Upload a screenshot or image of a puzzle to import it into akari-helper
